# إعداد Admin/Auth للوحة الأدمن

تم تجهيز لوحة الأدمن لتعمل عبر Firebase Authentication بالبريد وكلمة المرور.

## إنشاء أول أدمن

1. افتح Firebase Console.
2. من Authentication > Sign-in method فعّل Email/Password.
3. من Authentication > Users أنشئ مستخدم أدمن، مثال: `admin@example.com`.
4. انسخ UID الخاص بالمستخدم.
5. من Firestore أنشئ document داخل collection باسم `admins`، واجعل اسم المستند هو UID.
6. أضف الحقول التالية:

```json
{
  "name": "Main Admin",
  "email": "admin@example.com",
  "status": "active",
  "role": "admin"
}
```

## فتح حساب تجريبي من لوحة الأدمن

بعد الدخول إلى `admin/admin.html`:

- رقم الحساب 7 أرقام: `3014021`
- رقم الحساب الكامل 16 رقم: اتركه فارغًا ليتم توليده تلقائيًا، أو أدخل `0123030140210001`
- الاسم: `محمد فيصل البلولة محمد`
- الهاتف: `249912345678`
- الرصيد: `1000000`
- كلمة المرور: `1234`

ثم اضغط **فتح الحساب**.

## ملاحظة أمان مهمة

التطبيق الحالي يستخدم Firestore مباشرة للتحويلات، لذلك القواعد تسمح لمستخدم التطبيق الموقّع Anonymous بتحديث الرصيد أثناء التحويل.
للحماية البنكية الكاملة يجب نقل عمليات التحويل إلى API/Cloud Functions حتى لا يستطيع العميل تعديل الأرصدة مباشرة.
